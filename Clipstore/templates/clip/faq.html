OnlinePaste.in is an 'internet clipboard' or 'online pasteboard' which allows you to easily share text using the internet.

Features:

  1. Simple interface: A easy to use interface which allows one to create clips.
  2. Password protected clips - Password protected clips can be created for free.
  3. Custom keyword: It generates random clip ids for every clip but also allows you to choose your own keyworld.

Use cases:

  1. Sharing instructions with large audience with a link - Say a hackfest, classroom, events
  2. Moving information from one computer to another quickly where other means of access like email, messenger are unavailable.
  3. Sending large information via traditional means like SMS - A clip can be created and only the link can be sent.

Frequently Asked Questions

We love open source and believe in giving credit to them when used. The following components have been used:
  • Django - Web Framework
  • Bootstrap - For styling
  • Clipboard.js - To enable easy copying of the link and text
  • Fontawesome - Icons
  • sjcl.js - JavaScript cypto library to perform in browser encryption and decryption

To keep things simple and secure, we use a JavaScript crypto library to perform encryption and decryption in the browser itself. When the user enters a password, the text is encrypted in the browser and sent to the server. Since we perform the crypto operations in the browser, we do not need the password in the server and so they are not even sent to the server.
This means that we have no way of knowing what is encrypted and also means we cannot retrieve the text in case the password is lost.
In short: Passwords are never stored on the server!

During encryption, we also encrypt a well known text associated with each clip, in this case the clip url and store it in the db.
When the user tries to access a encrypted clip and enters a password, we try to decrypt this encrypted url first. If the decryption succeeds and the decrypted url matches the clip url, we will then try to decrypt the actual text using the password.
